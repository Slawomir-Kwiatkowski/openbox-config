<h2>My openbox config</h2>
<p>Applications run maximized by default, without window decorations.</p>
<ul>
  <li>Meta key + digit (default: 1 to 4) --- switch to desktop</li>
  <li>Meta key + Left --- switch to desktop on the left</li>
  <li>Meta key + h --- switch to desktop on the left</li>
  <li>Meta key + Right --- switch to desktop on the right</li>
  <li>Meta key + l --- switch to desktop on the right</li>
  <li>Meta key + Shift key + digit (default: 1 to 4) --- move app to desktop</li>
  <li>Meta key + Shift + Left --- move app to desktop on the left</li>
  <li>Meta key + Shift + h --- move app to desktop on the left</li>
  <li>Meta key + Shift + Right --- move app to desktop on the right</li>
  <li>Meta key + Shift + l --- move app to desktop on the right</li>
  <li>Meta key + Space bar --- run Runner app</li>
  <li>Meta key + Esc key --- close system</li>
  <li>Meta key + q --- close app</li>
  <li>Meta key + i --- run web browser (Brave broser) on WWW desktop</li>
  <li>Meta key + f --- run file browser (Pcmanfm-qt) on TOOLS desktop</li>
  <li>Meta key + c --- run Visual Studio Code on DEV desktop</li>
  <li>Meta key + n --- run nvim</li>
  <li>Meta key + s --- open scrapbook file with nvim</li>
  <li>Meta key + Return/Enter key --- run Qterminal on SYS desktop</li>
  <li>Meta key + Ctrl key + Up key --- increase screen brightness</li>
  <li>Meta key + Ctrl key + Down key --- decrease screen brightness</li>
  <li>Meta key + F9 key (volume up on my keyboard) --- increase sound volume</li>
  <li>Meta key + F10 key (volume down on my keyboard) --- decrease sound volume</li>
  <li>Meta key + F11 key (mute sound on my keyboard) --- mute sound</li>
  <li>PrtSc key --- make screenshot</li>  
</ul>
<p>No tiling of course, but all I need is tiling the terminal, which I do using qterminal's keyboard shortcuts, e.g. Alt + h --- horizontal tiling, Alt + v --- vertical tiling, Alt + q --- closes subterminal (Meta + q --- closes all subterminals), Alt + arrow keys --- cycles between subterminals.</p>
