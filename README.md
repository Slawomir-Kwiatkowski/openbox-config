<h2>My openbox & qterminal config</h2>
<p>Applications run maximized by default, without window decorations.</p>
<ul>
  <li>Super + digit (1 to 5) --- switch to chosen desktop</li>
  <li>Super + Numpad (1 to 5) --- switch to chosen desktop</li>
  <li> Super + Left --- switch to desktop on the left</li>
  <li> Super + h --- switch to desktop on the left</li>
  <li>Super + Right --- switch to desktop on the right</li>
  <li>Super + l --- switch to desktop on the right</li>
  <li>Super + Shift + digit (1 to 5) --- move app to chosen desktop</li>
  <li>Super + Shift + Numpad (1 to 5) --- move app to chosen desktop</li>
  <li>Super + Shift + Left --- move app to desktop on the left</li>
  <li>Super + Shift + h --- move app to desktop on the left</li>
  <li>Super+ Shift + Right --- move app to desktop on the right</li>
  <li>Super + Shift + l --- move app to desktop on the right</li>
  <li>Super + Spacebar --- run Runner app (app finder)</li>
  <li>Super + Esc --- close system</li>
  <li>Super + q --- close app</li>
  <li>Super + i --- run web browser (Brave broser) on WWW desktop & focus</li>
  <li>Super + p --- run file manager (Pcmanfm-qt) on TOOLS desktop & focus</li>
  <li>Super + f --- run file manager (LF) on TOOLS desktop & focus</li>
  <li>Super + c --- run Visual Studio Code on DEV desktop</li>
  <li>Super + n --- run nvim</li>
  <li>Super + s --- open scrapbook file with nvim</li>
  <li>Super + Enter --- run Qterminal on SYS desktop</li>
  <li>Super + d --- toggle Qterminal in dropdown mode (on every workspace)</li>
  <li>Super + Ctrl + Up --- increase screen brightness</li>
  <li>Super + Ctrl + Down --- decrease screen brightness</li>
  <li>Super + F9 (volume up) --- increase sound volume</li>
  <li>Super + F10 (volume down) --- decrease sound volume</li>
  <li>Super + F11 (mute sound) --- mute sound</li>
  <li>Print --- make screenshot</li>  
</ul>
<p>No tiling of course, but qterminal subterminals.</p>
<ul>
  <li>Alt + v --- tile vartically</li>
  <li>Alt + b --- tile horizontally</li>
  <li>Alt + h, j, k, l --- move across tiles (like in vim)</li>
  <li>Alt + q --- close tile (Super + q to close all tiles)</li>
</ul>
